# Обсидиановая обезьяна

**Описание:** Скульптура обезьяны из обсидиана, обнаруженная в Мексике. Отличается идеально круглой формой тела, достигнутой без использования гончарного круга -- технологии, которая не была известна в Мesoамерике.

**Книги:**
- [[13-meksika-bez-krivyh-zerkal]]

**Ключевые факты:**
- Материал: обсидиан (硬度 5-6 по шкале Мооса)
- Форма: идеально круглая
- Технология: обработка без гончарного круга
- Гончарный круг не был известен в Мesoамерике
- Для достижения такой круглости необходим токарный станок
- Хранится в Национальном музее антропологии в Мехико
- Свидетельство высокотехнологичной обработки

**Связанные сущности:**
- [[Национальный музей антропологии (Мехико)]] -- место хранения
- [[13-meksika-bez-krivyh-zerkal]] -- книга, описывающая артефакт
