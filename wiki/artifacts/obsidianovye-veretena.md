# Обсидиановые "веретёна"

**Описание:** Артефакты из обсидиана, обнаруженные в Мексике. Имеют форму веретён с тонкими стенками (1 мм), обработанных на токарном станке. Уникальность: обсидиан слишком хрупок для ручной обработки с такой точностью.

**Книги:**
- [[books/13-meksika-bez-krivyh-zerkal]]

**Ключевые факты:**
- Материал: обсидиан (硬度 5-6 по шкале Мооса)
- Форма: веретёнообразная
- Стенки: толщиной 1 мм
- Технология: токарная обработка
- Обсидиан слишком хрупок для ручной обработки
- Необходим точный контроль скорости вращения
- Хранятся в Национальном музее антропологии в Мехико
- Свидетельство применения токарных технологий

**Связанные сущности:**
- [[places/nacionalny-muzej]] -- место хранения
- [[books/13-meksika-bez-krivyh-zerkal]] -- книга, описывающая артефакты
