# Керн Питри N.7

**Описание:** Гранитный керн (образец), обнаруженный Флиндерсом Питри при раскопках в Асуане. Содержит чёткие спиральные следы сверления, свидетельствующие о применении технологии трубчатого сверления с высокой скоростью подачи.

**Книги:**
- [[12-tsivilizaciya-bogov-egipta]]

**Ключевые факты:**
- Обнаружен Флиндерсом Питри при раскопках в Асуане
- Материал: гранит (硬度 7 по шкале Мооса)
- Содержит чёткие спиральные следы сверления
- Расстояние между витками спирали: ~0,5 мм
- Эта скорость подачи несовместима с ручным буриением
- Измерения с помощью акселерометра (Данн): скорость подачи ~0,5 мм/оборот
- Медные буры с кварцевым песком не могут обеспечить такую скорость
- Предположительно использовался ультразвуковой бур (20 кГц) -- гипотеза Данна
