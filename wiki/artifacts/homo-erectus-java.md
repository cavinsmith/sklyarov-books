# Homo erectus Java

Окаменелые останки вида Homo erectus, обнаруженные на острове [[places/java|Ява]], ставшие одним из ключевых доказательств существования ранних человекообразных существ в Юго-Восточной Азии.

## Основные сведения

- Первоначально описаны Эженом Дюбуа в 1891 году как «Питекантроп»
- Обнаружены у реки Solo на острове [[places/java|Ява]]
- Датировка: от 1,5 млн до 108 тыс. лет назад
- Включают черепа, зубы и фрагменты скелета
- Ставят под вопрос традиционную модель «走出 Африки»
