# Саккара

**Описание:** Комплекс в Египте с ступенчатой пирамидой Джосера. Коллекция каменных сосудов с признаками механической обработки (токарная обработка).

**Книги:**
- [[24-opasnoe-nasledie]]
- [[12-tsivilizaciya-bogov-egipta]]

**Ключевые факты:**
- Ступенчатая пирамида Джосера -- древнейшая из известных каменных монументальных построек (XXVII в. до н.э. по традиционной датировке)
- Архитектор -- Имхотеп (один из первых известных по имени архитекторов)
- Коллекция каменных ваз и сосудов с обработанной внутренней поверхностью
- Сосуды выполнены из твёрдых пород камня (гранит, базальт, диорит)
- Следы токарной обработки -- концентрические борозды, характерные для вращательной механической обработки
- Точность обработки указывает на применение технологий, не соответствующих «каменному веку»
- Аналогичные артефакты обработки камня обнаружены и на [[Плато Гиза|плато Гиза]]

**Связанные сущности:**
- [[Плато Гиза]] -- соседний комплекс с аналогичными свидетельствами механической обработки
- [[Пирамиды Гизы]] -- расширенная статья о гизском комплексе
- [[Египет древний]] -- обзорная статья о древнем Египте
- [[Петри Флиндерс]] -- исследователь, зафиксировавший следы механической обработки
