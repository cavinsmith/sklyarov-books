# Филиппинская плита

**Описание:** Одна из самых маленьких литосферных плит Земли, расположенная в [[Филиппинское море|районе Филиппинского моря]].

**Книги:**
- [[Миф о Потопе: расчеты и реальность (01-mif-o-potope)]]

**Ключевые факты:**
- Значительно меньше любой другой плиты на Земле
- Имеет наклон в направлении с юго-востока на северо-запад
- Вместе с [[Плита Скота]] является одной из двух наименьших плит
- Один из аргументов в пользу гипотезы падения метеорита в [[Филиппинское море]]

**Связанные сущности:**
- [[Филиппинское море]] -- район расположения
- [[Плита Скота]] -- единственная плита, сопоставимая по размеру
- [[Всемирный Потоп]] -- метеорит, создавший впадину
- [[Падение метеорита в Филиппинское море]] -- гипотетическое событие
