# Филиппинская плита

**Описание:** Одна из самых маленьких литосферных плит Земли, расположенная в [[places/phippine-sea|районе Филиппинского моря]].

**Книги:**
- [[books/01-mif-o-potope]]

**Ключевые факты:**
- Значительно меньше любой другой плиты на Земле
- Имеет наклон в направлении с юго-востока на северо-запад
- Вместе с [[places/plita-skota]] является одной из двух наименьших плит
- Один из аргументов в пользу гипотезы падения метеорита в [[places/phippine-sea]]

**Связанные сущности:**
- [[places/phippine-sea]] -- район расположения
- [[places/plita-skota]] -- единственная плита, сопоставимая по размеру
- [[events/vsemirnyi-potop]] -- метеорит, создавший впадину
- [[events/padenie-meteorita]] -- гипотетическое событие
