# Лельчицы

**Описание:** Месторождение бурого угля в Беларуси. Данные по содержанию серы в угле из этого месторождения используются для критики биогенной теории происхождения угля.

**Упоминается в:** [[books/08-istoriya-zemli-bez-kamenno|История Земли без Каменноугольного периода]]

**Ключевые факты:**
- Месторождение бурого угля на территории Беларуси
- Аналитические данные по составу угля из Лельчиц показывают повышенное содержание серы
- Сера в составе угля не находит удовлетворительного объяснения в рамках биогенной теории
- Если уголь образован из растительных остатков, содержание серы должно быть минимальным
- Повышенное содержание серы подтверждает [[concepts/abiogennoe-proishozhdenie-nefti|abiогенную]] теорию происхождения угля
