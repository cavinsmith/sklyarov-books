# Трес-Сапотес

**Описание:** Ольмекский археологический объект в мексиканском штате Веракрус. Известен Стелой 3 с датой по Длинному Счёту (32 г. до н.э.) -- одним из самых ранних известных использований системы Long Count.

**Книги:**
- [[books/13-meksika-bez-krivyh-zerkal]]

**Ключевые факты:**
- Расположен в штате Веракрус, Мексика
- Известен Стелой 3 -- каменным монументом с датой по системе Длинного Счёта
- Дата на Стеле 3: 32 г. до н.э. (7.16.6.16.18 в системе Long Count)
- Это одно из самых ранних свидетельств использования системы Long Count
- Найдены также гигантские базальтовые головы
- Стела была обнаружена Мэттом Стройоном в 1939 году
- Наличие Длинного Счёта у ольмеков указывает на глубокую древность этой системы счисления
