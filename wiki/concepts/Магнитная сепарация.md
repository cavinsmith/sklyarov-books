# Магнитная сепарация

Метод разделения материалов с помощью магнитного поля. В работах [[people/andrey-sklyarov|Андрея Склярова]] рассматривается как один из технологических процессов, который мог быть известен [[concepts/Древние боги|«богам»]].

## Основные положения

- Метод разделения материалов на основе их магнитных свойств
- Используется в металлургии и горнодобывающей промышленности
- Может объяснять определённые археологические находки
- Связь с [[concepts/Металлы — дар небесных богов|технологиями обработки металлов]]

## Применение

- Изготовление магнитов из природных материалов
- Разделение руды и породы
- Очистка металлов
- Возможное использование в древних технологиях
