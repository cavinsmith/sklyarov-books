# Гемоцианин

Медесодержащий дыхательный пигмент, обеспечивающий транспорт кислорода у многих беспозвоночных организмов. В отличие от [[concepts/gemoglobin|гемоглобина]], содержащего железо, [[concepts/gemocianin|гемоцианин]] использует ионы меди для связывания молекул кислорода.

## Структура и функции

- Белковая молекула с двумя ионами меди в активном центре
- Медь связывает один молекулу O₂: Cu⁺ + O₂ → Cu²⁺-O₂
- При насыщении кислородом даёт голубой цвет ([[concepts/golubaya-krov|голубая кровь]])
- При отдаче кислорода становится бесцветной
- Содержится в гемолимфе беспозвоночных

## Сравнение с гемоглобином

| Свойство | [[concepts/gemocianin|Гемоцианин]] | [[concepts/gemoglobin|Гемоглобин]] |
|---|---|---|
| Металл | Медь (Cu) | Железо (Fe) |
| Цвет крови | Голубой | Красный |
| Молекула носитель | Белок (без гема) | Гем (порфириновое кольцо) |
| Буферная ёмкость | Ниже | Выше |
| Распространённость | Беспозвоночные | Позвоночные |

## Ключевая роль в теории Склярова

В [[books/05-rodina-bogov|«Какова ты, Родина Богов?]] [[concepts/gemocianin|гемоцианин]] является центральным элементом аргументации:

- Боги использовали [[concepts/gemocianin|гемоцианин]] для дыхания — следовательно, на [[places/rodnaya-planeta-bogov|родной планете]] медь была преобладающим металлом
- Слабая буферная ёмкость [[concepts/gemocianin|гемоцианина]] (по сравнению с [[concepts/gemoglobin|гемоглобином]]) объясняет потребность богов в [[concepts/soma-napitok-bogov|алкогольных напитках]] для нейтрализации CO₂
- Переход на [[concepts/gemoglobin|гемоглобин]] (железосодержащий) мог произойти после адаптации к земным условиям с преобладанием железа
