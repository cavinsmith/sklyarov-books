# Цитохром C

Белок, участвующий в клеточном дыхании (транспорт электронов в цепи переноса электронов митохондрий). Используется как один из основных «молекулярных часов» для определения времён эволюционного расхождения видов благодаря высокой степени сохранности структуры у разных организмов.

## Основные сведения

- Универсальный белок, присутствующий практически у всех аэробных организмов -- от бактерий до человека
- Играет ключевую роль в клеточном дыхании: переносит электроны в митохондриях
- Очень консервативный белок: аминокислотная последовательность почти неизменна на протяжении сотен миллионов лет
- Именно высокая степень консервативности делает его пригодным для [[concepts/molekulyarnye-chasy|молекулярных часов]]
- Человек и шимпанзе различаются всего в одной аминокислоте из 104 в молекуле цитохрома C

## Ключевые факты

- Сравнение цитохрома C у разных видов позволяет определить степень их родства и время расхождения
- Различия в аминокислотных последовательностях накапливаются приблизительно пропорционально времени
- По данным цитохрома C: человек и шимпанзе -- очень близкие родственники, человек и дрожжи --远 более далекие
- Молекулярные часы на основе цитохрома C дают оценки времени расхождения, которые не всегда совпадают с палеонтологическими данными
- Цитохром C -- один из наиболее изученных белков в эволюционной биологии

## Упоминание в книге «Генетический код человечества»

В книге Андрея Склярова цитохром C используется как ключевой пример противоречий между данными молекулярной биологии и классической [[concepts/teoriya-evolyucii|теорией эволюции]]:

- Анализ цитохрома C показывает, что [[concepts/molekulyarnye-chasy|молекулярные часы]] дают оценки расхождения видов, отличающиеся от палеонтологических
- Расхождение человека и обезьяны по цитохрому C указывает на более поздние сроки, чем предполагает классический [[concepts/evolycionizm|эволюционизм]]
- Степень различий в цитохроме C между человеком и другими видами не всегда согласуется с предсказаниями дарвиновской теории
- Автор приводит данные о цитохроме C как пример несоответствия между генетическими и палеонтологическими оценками эволюционных сроков
- Цитохром C -- один из аргументов в пользу того, что эволюция человека могла иметь иную динамику, чем предполагает чистый эволюционизм
- Данные по цитохрому C и другим белкам согласуются с гипотезой о возможном внешнем вмешательстве в геном человека
