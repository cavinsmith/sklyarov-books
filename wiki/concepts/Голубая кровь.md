# Голубая кровь

Концепция, связанная с особенностью крови определённых существ. В работах [[people/Андрей Скляров|Андрея Склярова]] рассматривается как один из возможных признаков [[concepts/Палеоконтакт|контакта]] с [[concepts/Древние боги|«богами»]].

## Основные положения

- Термин «голубая кровь» используется для описания знати
- Могет связан с особенностью крови у некоторых групп населения
- Связь с [[concepts/Гемоцианин|гемоцианином]] вместо [[concepts/Гемоглобин|гемоглобина]]
- Возможный признак генетической модификации
