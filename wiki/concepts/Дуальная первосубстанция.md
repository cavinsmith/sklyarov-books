# Дуальная первосубстанция

Концепция фундаментальной основы мироздания, описанная в работах [[people/Андрей Скляров|Андрея Склярова]] в контексте «физики духа».

## Основные положения

- Фундаментальная основа мироздания
- Двойственная природа: материя и дух
- Связь с [[concepts/Духовно-нематериальный мир|духовным миром]]
- Основа для понимания [[concepts/Пирамида души|структуры души]]

## Характеристики

### Двойственность
- Материальная составляющая
- Духовная составляющая
- Единство противоположностей

### Проявление
- В физическом мире
- В духовном мире
- В структуре души

## Значение

- Основа для понимания мироздания
- Связь с научными теориями
- Философское значение
