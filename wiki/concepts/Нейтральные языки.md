# Нейтральные языки

Концепция языков, которые могли использоваться для общения между [[concepts/Древние боги|«богами»]] и людьми, или между разными группами «богов». В работах [[people/andrey-sklyarov|Андрея Склярова]] рассматривается как один из аспектов [[concepts/Палеоконтакт|палеоконтакта]].

## Основные положения

- «Боги» могли использовать особые языки для общения
- Эти языки могли быть основой для священных текстов
- Связь с появлением письменности
- Лингвистические следы контакта в разных культурах

## Свидетельства

- Сходство священных терминов в удалённых культурах
- Лингвистические заимствования между несвязанными языками
- Мифы о «языке богов»
- Появление письменности как след контакта
