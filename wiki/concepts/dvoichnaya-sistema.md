# Двоичная система счисления

**Другие названия:** Двоичная запись,.Binary notation

**Описание:** [[concepts/pozizionnaya-zapis|Позиционная система счисления]] с основанием 2, в которой для записи чисел используются только два символа: 0 и 1. Согласно [[books/10-kompyuter-drevnego-kitaya|А. Склярову]], эта система была известна древним китайцам около 5000 лет назад и нашла отражение в структуре [[concepts/kniga-peremen|Книги Перемен]].

---

## Ключевые факты

- Каждая позиция в двоичном числе соответствует степени двойки (1, 2, 4, 8, 16, 32...)
- [[concepts/trigrama|Триграммы]] кодируют двоичные числа от 0 до 7 (3 разряда)
- [[concepts/geksagramma|Гексаграммы]] кодируют двоичные числа от 0 до 63 (6 разрядов)
- В системе И-Цзин: [[concepts/yang|ян]] (сплошная) = 1, [[concepts/yin|инь]] (прерванная) = 0
- Современная вычислительная техника основана на двоичной системе

---

## Связи

- [[concepts/kniga-peremen|Книга Перемен]] -- структура гексаграмм соответствует двоичным числам
- [[concepts/princzip-inverzii|Принцип инверсии]] -- чтение двоичного кода в обратном порядке
- [[concepts/princzip-dopolneniya|Принцип дополнения]] -- замена 0 на 1 и обратно
